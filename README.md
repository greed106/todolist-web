## 📌 项目简介

- **项目名称**：ToDoList 日程管理软件
- **开发模式**：前后端分离
- **任务管理模式**：采用“项目分类管理任务”的模式，每个任务归属于一个项目管理。
- **服务器支持**：使用由 [Microsoft Azure](https://azure.microsoft.com/) 提供的服务器。
- **部署**：前端静态页面部署在 [GitHub Pages](https://greed106.github.io/todolist-web/) 上。
